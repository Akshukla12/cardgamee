<!-- # cardgame

An interactive memory card game built with HTML, CSS, and JavaScript.

## Features
- Multiple difficulty levels (4x4, 6x6, 8x8)
- Score tracking
- Timer functionality
- Leaderboard
- Responsive design

## Technologies Used
- HTML5
- CSS3
- JavaScript
- Local Storage API

## How to Play
1. Select difficulty level
2. Click 'Start Game'
3. Match pairs of cards
4. Complete the game with highest score
